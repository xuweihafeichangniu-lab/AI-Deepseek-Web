# 云服务器部署指南 (Web版)

我已经为您生成了可用于云服务器部署的 Web 版本。所有代码已存放在新文件夹：
`d:\AIdeepseekOK6\AIdeepseekOK\AI deepseek chaoji\AI_deepseek_web`

## 目录结构
- `backend/`: Node.js Express 服务器（负责处理 API 签名和 AI 请求）。
- `frontend/`: React 前端源代码及构建后的生产文件（在 `dist` 目录）。

## 部署步骤

### 1. 准备环境
确保您的云服务器（Linux 或 Windows Server）已安装 **Node.js**。

### 2. 上传文件
将整个 `AI_deepseek_web` 文件夹上传到您的云服务器。

### 3. 安装依赖与启动
在服务器上执行以下指令：

```bash
# 进入后端目录
cd AI_deepseek_web/backend

# 安装依赖
npm install

# 启动服务器
node server.js
```

### 4. 访问系统
- 服务器启动后，您可以通过浏览器访问：`http://服务器公网IP:3000`
- 默认端口为 `3000`（如有需要可在 `backend/server.js` 中修改）。

## 注意事项
- **配置 API 密钥**：您可以在服务器的 `backend` 目录下新建 `.env` 文件，或者直接在启动后的网页设置页面输入密钥。
- **防火墙**：请确保您的云服务器防火墙开放了 `3000` 端口。
- **持久运行**：建议使用 `pm2` 来通过后台持久运行程序。
  ```bash
  npm install -g pm2
  pm2 start server.js --name binance-ai-bot
  ```
