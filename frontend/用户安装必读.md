# 🤖 SUPER AI 量化交易系统 - 用户安装与配置指南

欢迎使用 SUPER AI 量化交易系统。为了确保软件在您的电脑上顺利运行，请仔细阅读以下安装和配置说明。

---

## 💻 一、 电脑配置要求 (最低配置)

此软件基于 Electron 和现代前端技术开发，对电脑配置要求不高，但为了流畅运行 AI 分析和实时图表，建议：

*   **操作系统**: Windows 10 (64位) 或 Windows 11
*   **处理器 (CPU)**: Intel i5 第8代及以上 / AMD Ryzen 5 2000系列及以上
*   **内存 (RAM)**: 8GB 或以上 (建议 16GB)
*   **硬盘**: 至少 10GB 可用空间 (SSD 固态硬盘体验更佳)
*   **网络**: 必须具备稳定访问国际网络的能力 (需要连接币安 API 和 DeepSeek AI)

---

## 🛠️ 二、 环境配置 (源码运行必看)

如果您拿到的是**源代码**包，需要在你的电脑上搭建开发环境。如果您拿到的是 `.exe` 安装包，请跳过此步骤直接看第三章。

### 1. 安装 Node.js (核心运行环境)
*   前往 Node.js 官网: https://nodejs.org/zh-cn/
*   下载 **LTS 版本** (长期支持版，例如 v18.x 或 v20.x)。
*   安装时一路点击 "Next" (下一步) 即可，保持默认设置。
*   安装完成后，打开 CMD (命令提示符)，输入 `node -v`，如果有版本号显示 (如 `v20.11.0`)，说明安装成功。

### 2. 代码编辑器 (推荐)
*   推荐下载 **Visual Studio Code (VS Code)**: https://code.visualstudio.com/
*   这是一个免费且强大的代码查看和编辑工具。

---

## 🚀 三、 软件安装与启动

### 情况 A: 只有源代码 (Source Code)
1.  解压您购买的压缩包到一个没有中文路径的文件夹 (例如 `D:\SuperAI`).
2.  进入该文件夹，在地址栏输入 `cmd` 并回车，打开黑色的命令窗口。
3.  **安装依赖**:
    在窗口中输入以下命令并回车：
    ```bash
    npm install
    ```
    *这需要几分钟时间，请耐心等待直到进度条跑完。如果不动了，请检查网络。*

4.  **启动软件**:
    输入以下命令并回车：
    ```bash
    npm run electron:dev
    ```
    软件窗口稍后会自动弹出。

### 情况 B: 已有安装包 (.exe)
1.  直接双击安装程序 (`SuperAI-Setup.exe`)。
2.  按提示完成安装。
3.  点击桌面图标运行。

## ⚙️ 四、 关键配置 (新手必读)

软件启动后，**必须**完成以下配置才能工作：

### 1. 代理配置 (重中之重)
由于币安 (Binance) 和 DeepSeek 的服务器在海外，国内网络通常无法直接连接，您需要自备“魔法”工具 (VPN/代理软件)。

1.  打开您的代理软件 (Clash, v2rayN 等)。
2.  查看代理软件的 **本地端口 (Local Port)**，通常是 `7890`, `7897` 或 `10808`。
3.  确保开启 **允许局域网连接** 或 **TUN模式** (建议开启 Tun模式以获得最佳稳定性)。
4.  在 SUPER AI 软件中 -> **设置 (Settings)** -> **网络代理配置**。
5.  填入您的本地代理地址，例如：
    *   HTTP模式: `http://127.0.0.1:7897`
    *   SOCKS模式 (推荐): `socks5://127.0.0.1:7897`
    *(请将 7897 替换为您实际的代理端口)*

### 2. API Key 配置
1.  登录您的 **币安 Web 端**。
2.  申请一对新的 **API Key** 和 **Secret Key**。
    *   权限勾选: "读取数据" 和 "允许合约交易" (Enable Futures)。
    *   **注意**: 这是一个交易软件，请务必保管好您的 Key，不要发给任何人。
3.  登录 **DeepSeek 开放平台** (platform.deepseek.com) 申请 API Key。
4.  在 SUPER AI 软件 -> **设置** 中依次填入。
5.  点击 **"验证"** 按钮。
    *   系统会自动保存您的配置。
    *   如果显示绿色成功提示，恭喜您，配置完成！此时可以开始使用 AI 自动交易功能。

## ❓ 常见问题

**Q: 点击验证提示 "Signature not valid" (-1022)?**
A: 您的 API Secret 填错了，或者复制时带了空格。请在币安重新生成一个新的 Key/Secret 对。

**Q: 提示网络连接失败?**
A: 请检查代理软件是否开启，端口是否填写正确。此时可以尝试把代理地址前缀改为 `socks5://` 试试。

**Q: 软件界面一片空白?**
A: 请检查是否安装了 Node.js，或者尝试重新运行 `npm install`。
